## 快速说明

`AutoMailLogin` 是一个基于 `Paper` 的 Minecraft 邮箱认证插件，支持：

- 邮箱注册与验证码验证
- 密码登录与密码重置
- 邮箱二次验证（2FA）
- `SQLite / MySQL` 双存储
- 基础认证 GUI
- 管理员状态查询与审计日志
